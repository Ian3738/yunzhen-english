# 昀蓁學苑

杜昀蓁老師的高中**英文**與**國文**互動練習集合（昀蓁學苑），共 **70 個**練習，從原本的 CodePen 作品整理、分類後部署於 GitHub Pages。設計採 Art Deco 風格（昀蓁老師指定）。

🔗 **網站**：https://ian3738.github.io/yunzhen-english/

## 內容

| 科目 | 練習數 | 主題 |
|------|-------|------|
| 英文 | 55 | 課本單字與課文、單字／常見字、片語、文法（時態與動詞、be 動詞、句型、課程與總複習）、考試準備 |
| 國文 | 15 | 課文、諸子古文、詞選新詩選文 |

## 結構（多頁式）

```
index.html           沉浸式入口（金色大門開啟 → 學苑大廳 → 選擇英文/國文學習廳）
english.html         英文：年級分頁籤（高一/高二/高三/不分年級）→ 主題分類 → 練習卡
chinese.html         國文：年級分頁籤（高二/不分年級）→ 主題分類 → 練習卡
assets/site.css      共用樣式（Art Deco 深色主題）
assets/*.js          入口門動畫、年級分頁與搜尋
assets/*.jpg         Nano Banana Pro 生成的插畫（大廳、英文房、國文房）
apps/<id>/index.html 每一個互動練習（自 CodePen 原樣保存）
```

每個練習都是獨立、自包含的 HTML，可單獨開啟。

## 使用

開啟入口頁 → 推門進入 → 選英文或國文 → 用上方年級頁籤切換，或搜尋框找特定練習。
一次只看一個年級，頁面簡短好瀏覽；同一年級內再依主題分類。
