# 昀蓁老師的學習網

杜昀蓁老師的高中**英文**與**國文**互動練習集合，共 **70 個**練習，從原本的 CodePen 作品整理、分類後部署於 GitHub Pages。

🔗 **網站**：https://ian3738.github.io/yunzhen-english/

## 內容

| 科目 | 練習數 | 主題 |
|------|-------|------|
| 英文 | 55 | 課本單字與課文、單字／常見字、片語、文法（時態與動詞、be 動詞、句型、課程與總複習）、考試準備 |
| 國文 | 15 | 課文、諸子古文、詞選新詩選文 |

## 結構

```
index.html          首頁（分類、篩選、搜尋）
apps/<id>/index.html 每一個互動練習（自 CodePen 原樣保存）
```

每個練習都是獨立、自包含的 HTML，可單獨開啟。

## 使用

直接開啟首頁，依科目／主題瀏覽，或用搜尋框找特定練習名稱。
