🤷 sometimes 有時候
📊 usually 通常
🚫 never 從不
✏️ Part 2: 看中文寫英文

2. 今天 →
3. 今晚 →
4. 昨天 →
5. 明天 →
6. 經常 →
7. 總是 →
8. 有時候 →
9. 通常 →
10. 從不 →
📝 Part 3: 看英文寫中文
1. now →
2. today →
3. tonight →
4. yesterday →
5. tomorrow →
6. often →
7. always →
8. sometimes →
9. usually →
10. never →
💡 文法應用:
這些時間副詞常用在不同時態:
• now, today, tonight → 現在式/現在進行式
• yesterday → 過去式
• tomorrow → 未來式
• often, always, sometimes, usually, never → 一般現在式
